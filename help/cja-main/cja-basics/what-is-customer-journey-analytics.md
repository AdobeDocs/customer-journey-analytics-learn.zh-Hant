# 什麼是 Customer Journey Analytics？

Customer Journey Analytics (CJA) 可讓您使用 Adobe Experience Platform 中可用的任何資料跨管道分析客戶的旅程，同時利用 Analysis Workspace 的強大功能取得快速分析。
