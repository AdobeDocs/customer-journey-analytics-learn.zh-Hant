# 了解 Customer Journey Analytics

Adobe Customer Journey Analytics(CJA)是一種在Adobe Experience Platform之上構建的應用程式服務。 其將多樣化的分析工具，即 Analysis Workspace 帶至平台，讓您可對您的任何平台資料集執行多通道分析。
