# 自由表格中的多個維度欄

使用CJA中的多個Dimension欄功能，讓您的自由表格功能更強大。 除了標準維度劃分外，您也可以並排分析維度資料，以建立關聯配對，藉此更好地呈現和瞭解您的資料。 進階排序功能也提供更大的彈性，讓您能夠操控表格，依您想要的方式顯示資料！ 如需詳細資訊，請瀏覽[檔案。](https://experienceleague.adobe.com/zh-hant/docs/analytics-platform/using/cja-workspace/visualizations/freeform-table/freeform-table-multidimensions){target="_blank"}
