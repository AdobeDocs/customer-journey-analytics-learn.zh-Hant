# 使用Analysis Workspace中的元件下拉選單

使用元件下拉式選單從相容元件清單中挑選元件，以建置您的視覺效果。 如果您找不到自己所需的項目，您也可以直接從下拉式選單建立新的元件。 如果您有更進階的使用案例，這些空間也會繼續做為下拉區域作用。
