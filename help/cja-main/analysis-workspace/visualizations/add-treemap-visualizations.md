# 將[!UICONTROL 樹狀圖]視覺效果新增至[!DNL Customer Journey Analytics]中的[!DNL Analysis Workspace]個專案

瞭解如何在[!DNL Customer Journey Analytics]的[!DNL Analysis Workspace]個專案中新增並設定[!UICONTROL 樹狀圖]視覺效果。 此視覺效果會將階層資料建立為一組巢狀矩形，讓您輕鬆檢視在其他情況下很難發現的模式。

>[!VIDEO](https://video.tv.adobe.com/v/3480267/?captions=chi_hant&quality=12&learn=on)

如需[!UICONTROL 樹狀圖]視覺效果的詳細資訊，請瀏覽[檔案](https://experienceleague.adobe.com/docs/analytics-platform/using/cja-workspace/visualizations/treemap.html?lang=zh-Hant){target="_blank"}。
