瞭解如何使用Customer Journey Analytics中的快速深入分析面板快速輕鬆回答業務問題。 此面板適用於新使用者或進階分析人員，可讓您直觀地實驗維度、量度、視覺效果和區段，為您的分析產生最佳元件組合。
