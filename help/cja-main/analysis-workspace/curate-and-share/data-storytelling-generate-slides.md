# 資料Storytelling：產生投影片簡報

探索Adobe Customer Journey Analytics中強大的[!DNL Data Storytelling]功能，該功能使用創作AI將您的Analysis Workspace專案轉換為專業的PowerPoint簡報，並包含精選的視覺效果、關鍵分析和引人入勝的敘述。 此節省時間的工具可協助您快速與利害關係人分享客戶歷程趨勢、轉換推動因素和可操作的建議。

如需詳細資訊，請審閱 Content Analytics [文件](https://experienceleague.adobe.com/zh-hant/docs/analytics-platform/using/cja-workspace/curate-share/generate-slides){target="_blank"}。
