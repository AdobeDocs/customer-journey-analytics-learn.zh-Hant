# 建立自訂範本

了解如何在 Customer Journey Analytics 中建立自訂範本。 這支援針對特定業務目標量身打造的報表設定，例如分析行銷管道效能和使用者參與趨勢。 此影片示範如何設定具有「管道效能概述」和「時間分段分析」等面板的專案、儲存為範本，並讓組織可在Analysis Workspace中存取該專案。

如需詳細資訊，請瀏覽本[文件](https://experienceleague.adobe.com/zh-hant/docs/analytics-platform/using/cja-workspace/templates/create-templates){target="_blank"}。
