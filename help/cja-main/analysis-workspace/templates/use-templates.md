# 使用範本

瞭解如何在[!DNL Customer Journey Analytics]中使用範本以簡化分析和報告。 範本是預先建立的設定，用於常見業務分析，例如網站造訪、搜尋關鍵字和訪客持續時間。 本影片會引導您在Analysis Workspace中選取、預覽和自訂範本，以建立量身打造的專案。

如需詳細資訊，請瀏覽本[文件](https://experienceleague.adobe.com/zh-hant/docs/analytics-platform/using/cja-workspace/templates/use-templates){target="_blank"}。
