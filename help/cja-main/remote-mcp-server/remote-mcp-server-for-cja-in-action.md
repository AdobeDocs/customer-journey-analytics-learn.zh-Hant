# [!DNL Customer Journey Analytics]的遠端MCP伺服器正在執行中

瞭解如何將Claude、ChatGPT或Cursor等AI工具連線至Adobe Customer Journey Analytics (CJA) MCP伺服器，以探索資料檢視、執行報告並使用自然語言產生可用於執行人員的KPI深入分析。 此示範會展示遠端MCP伺服器如何在Adobe Experience Platform中啟用可擴充、控管的分析工作流程和自動化報表。

如需詳細資訊，請審閱 Content Analytics [文件](https://developer.adobe.com/analytics-mcp/docs/cja/){target="_blank"}。
