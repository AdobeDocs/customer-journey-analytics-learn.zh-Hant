# 在Customer Journey Analytics的衍生欄位中使用math函式調整量度

在此影片教學中，探索使用數學函數在 Customer Journey Analytics 中建立衍生欄位的潛力。 瞭解如何透過從總計中移除不需要的元件（例如稅金）來調整量度，並瞭解使用衍生欄位相對於計算量度的優點，包括獨佔的資料檢視功能，例如去重複化、分組和歸因。 獲得彈性來建立符合您分析需求的量身打造量度，進而增強您的資料導向式決策功能。
