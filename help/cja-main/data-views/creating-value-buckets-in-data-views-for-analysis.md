# 在[!UICONTROL 資料檢視]中建立值區以供分析

在 Customer Journey Analytics 中設定[!UICONTROL 資料檢視]時，您可以設定要集中到已定義值區內的數值維度，這樣您就可以將表格可用的項目數量限制在數字範圍 (超酷)。此影片將逐步說明此功能的設定。
