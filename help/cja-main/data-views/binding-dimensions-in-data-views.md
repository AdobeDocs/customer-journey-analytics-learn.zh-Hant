# 在資料檢視中繫結維度

繫結維度是 Customer Journey Analytics 中的一個現有功能，它讓您能夠取用一個維度，並將其連接到另一個維度，以獲得更精細的持續性配置。 我們經常遇到以下狀況：在使用者的整個歷程中接收不同的值，而且需要決定如何配置這些成功量度。 繫結維度是協助處理一些重要情境的方式。
