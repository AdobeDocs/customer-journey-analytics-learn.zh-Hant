# [!UICONTROL 資料檢視]中的[!UICONTROL 歸因]設定

每個維度都應該有一個分配設定 (以確認哪個數值會獲得轉換點數) 和一個期限設定 (數值會持續多久)，讓系統知道如何為維度值分配數值。同樣地，量度也應該有一個歸因模型和回顧窗口集。此影片會逐步引導您設定這些非常重要的設定。
