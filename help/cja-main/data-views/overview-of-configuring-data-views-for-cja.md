# 為 Customer Journey Analytics 設定資料檢視的概觀

在 Customer Journey Analytics 中，資料檢視類似於 Adobe Analytics 中的虛擬報表套裝；資料檢視可讓您設定進入 CJA 的資料，以便這些資料可以在報告和分析中發揮最大的用處。 這段影片提供為 Customer Journey Analytics 設定資料檢視的概觀。
