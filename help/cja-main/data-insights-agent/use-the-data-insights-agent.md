# 使用 [!DNL Data Insights Agent]

使用自然語言提示分析資料，產生自由格式表格、視覺化圖表及資料比較，降低對於資料分析師的依賴。 此 AI 驅動的代理可針對趨勢和效能提供快速洞察，從而產生可操作洞察，並馬上與利害關係人分享。
