# [!DNL Data Insights Agent]個使用案例

使用自然語言查詢、即時視覺效果和自由表格來簡化分析。 提升行銷策略開發、行銷活動管理和業務績效報告的效率。 此AI支援解決方案可讓行銷人員即時獲得深入分析，並釋放分析人員以執行策略任務、增強客戶體驗及主動決策。
