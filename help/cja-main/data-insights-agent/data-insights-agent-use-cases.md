# [!DNL Data Insights Agent] 使用案例

使用自然語言查詢、即時視覺效果和自由格式表格，簡化分析過程。 大幅提升行銷策略開發、行銷活動管理，以及業務績效報告的效率。 此 AI 驅動的解決方案可讓行銷人員即時獲取洞察分析，並讓分析師能夠專注於執行策略任務，藉此增強客戶體驗及強化主動決策的能力。
