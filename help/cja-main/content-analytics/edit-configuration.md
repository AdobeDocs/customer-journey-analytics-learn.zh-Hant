# [!DNL Content Analytics] — 編輯設定

調整名稱、資料檢視和體驗擷取設定，有效編輯[!DNL Content Analytics]設定。 此動作能自動更新量度，並透過 Content Analytics 標記擴充功能執行微調，讓洞察達到最佳化。

如需詳細資訊，請審閱[文件](https://experienceleague.adobe.com/zh-hant/docs/analytics-platform/using/content-analytics/configuration/guided){target="_blank"}。
