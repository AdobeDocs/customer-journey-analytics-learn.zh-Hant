# 善用日期欄位支援

日期欄位新增使用 Customer Journey Analytics 中日期和日期時間欄位的功能。您可以選取日期或日期時間欄位的顯示方式，以及在包含 / 排除功能中使用日期特定邏輯運算子，以篩選顯示的資料。
