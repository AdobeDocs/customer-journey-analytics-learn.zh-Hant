# 利用日期欄位支援

「日期」欄位支援新增了在Customer Journey Analytics中使用日期和日期時間欄位的能力。 您可以選取日期或日期時間欄位的顯示方式，以及在包含/排除功能中使用日期特定邏輯運算子來篩選顯示的內容。
