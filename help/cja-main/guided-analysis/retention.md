# 保留

探索Customer Journey Analytics中的&#x200B;**保留**。 了解如何使用開始和返回事件來有效測量使用者參與度和保留率。

瞭解如何使用開始和傳回事件來有效測量使用者參與度和保留率。 本教學課程也涵蓋如何調整不同業務模式的保留設定，無論您執行的是旅遊網站、電子商務平台或生產力工具。 取得使用保留資料以最佳化產品市場適用性並推動持續使用者參與的實用秘訣。
