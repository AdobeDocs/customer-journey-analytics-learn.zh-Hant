# 檢視、疑難排解和修改連線設定

連線管理員可讓您檢查連線資料集擷取的狀態。 當資料可用於在 Workspace 中開始分析時，此介面也會通知您。 了解如何識別因設定錯誤導致的資料差異、管理資料匯入日期以及從連線中移除資料集。
