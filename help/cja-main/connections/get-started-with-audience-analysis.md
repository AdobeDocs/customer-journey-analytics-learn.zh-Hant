# 開始使用對象分析

探索Audience Analysis如何讓行銷人員從Experience Platform中擷取受眾區段，並分析跨管道的真實客戶行為，以便更佳地進行促銷活動最佳化。 瞭解啟用受眾會員資格資料的簡單設定步驟、探索預先建立的範本，並找出可促進個人化、轉換和ROI的可操作深入分析。

如需詳細資訊，請審閱[文件](https://experienceleague.adobe.com/en/docs/analytics-platform/using/cja-connections/audience-analysis/audience-analysis-overview){target="_blank"}。
