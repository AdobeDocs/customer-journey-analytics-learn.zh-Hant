# 協助高階主管存取行動計分卡

協助高階主管使用 [!DNL Analytics] 儀表板行動應用程式來存取在 [!DNL Customer Journey Analytics] 中建立的行動計分卡。行動計分卡可讓使用者隨時存取關鍵效能量度和詳細檢視或解析。
