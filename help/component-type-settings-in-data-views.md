# 資料檢視中的元件類型設定

在Customer Journey Analytics的「資料檢視」中設定元件時，您可以選擇欄位應是量度或維度。 在許多情況下，字串應為維度，數字應為量度。 不過，有些絕佳的使用案例可供切換。 看看！
