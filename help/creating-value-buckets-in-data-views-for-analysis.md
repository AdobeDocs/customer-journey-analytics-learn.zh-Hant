# 在[!UICONTROL 資料檢視]中建立值貯體以進行分析

在Customer Journey Analytics中設定[!UICONTROL 資料檢視]時，您可以設定要集中到定義貯體的數值維度，以便將表格可用的項目數量限制在數字範圍（非常酷）。 此影片會逐步帶您了解此功能的設定。
