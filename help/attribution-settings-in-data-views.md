# [!UICONTROL 資料檢視中的歸因設定]

每個維度都應具備配置設定（以決定哪個值獲得轉換的評分）和到期設定（值持續存在的時間），讓系統知道如何將值指派給維度值。 同樣地，量度也應該有歸因模型和回顧期間集。 此影片會逐步帶您了解如何設定這些非常重要的設定。
