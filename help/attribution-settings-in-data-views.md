# [!UICONTROL 資料檢] 視中的 [!UICONTROL 歸因設定]

每個維度都應有配置設定（以決定哪個值會獲得轉換的評價）和過期設定（值持續多久），讓系統知道如何指派值給維度值。 同樣地，量度也應該有歸因模型和回顧視窗集。 此影片會逐步帶您設定這些重要設定。
